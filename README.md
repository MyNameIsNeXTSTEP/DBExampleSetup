# DBExampleSetup
A boilerplate project to prepare your dev DB setup locally

# Motivation

This tool allows you to do seceral things from the scratch by just ONE command:

1. Create a local docker container of desired db type (PostgreSQL, MySQL, etc.)
2. Prapare the migrations and seeds mechanics
3. Prepare connections variables
4. Get the ERD graph / DB dump

The goal is to get to the point where you can locally scaffold the preparations with ease.
Make your changes to the DB to establish a desired structure and give your team the data relationship files
